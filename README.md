Cavy is a dynamically-typed object-oriented general-purpose programming language, where everything is an object.

---
Hello world in Cavy:
```
System ctx {

  ^Main: * ctx {

    ^console->out: "Hello, world!".

    ^return: 0
  }.

}
```
